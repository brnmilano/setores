## Iniciando o projeto

```
    git clone https://github.com/brnmilano/setores
    cd setores
```

## Iniciando o front
Instala as dependencias para rodar o projeto.

```
    Instalar dependencias:
    yarn 
```

### Desenvolvimento

Inicia o desenvolvimento com atualização em tempo real no navegador.

```sh
yarn start 
```
